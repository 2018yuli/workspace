## 入门提示

## 如何使用

1. 安装 Slack 应用，连接正确的代码库和环境，并将 `@Codex` 添加到频道中。
2. 在线程中提及 `@Codex`，并清楚地提出请求、约束以及您想要的结果。
3. 打开任务链接，查看结果，如有需要再次处理，请在 Slack 中继续后续操作。

您可以在 [专门的指南](https://developers.openai.com/codex/integrations/slack) 中了解更多有关如何在 Slack 中使用 Codex 的信息。

## 提示

- 如果线程中没有足够的上下文或建议修复，请在您的提示中包含一些指导
- 通过在提示中提及项目或环境的名称，确保代码库和环境映射是正确的
- 确定请求的范围，以便 Codex 能够在没有第二次计划环节的情况下完成任务
- 如果您的项目是大型代码库，通过提及相关的文件或文件夹来指导 Codex 完成任务
