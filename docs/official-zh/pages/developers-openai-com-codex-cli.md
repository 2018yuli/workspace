Codex CLI 是 OpenAI 的编码代理，您可以从终端本地运行。它可以读取、修改并在您选择的目录中运行代码。
它是 [开源的](https://github.com/openai/codex)，并使用 Rust 构建，以实现速度和效率。

ChatGPT Plus、Pro、Business、Edu 和 Enterprise 计划包括 Codex。了解更多关于 [包含的内容](https://developers.openai.com/codex/pricing)。

### 运行

在终端中运行 Codex。它可以检查您的代码库、编辑文件并运行命令。

运行 Codex 命令

codex复制

第一次运行 Codex 时，系统会提示您登录。可以使用您的 ChatGPT 帐户或 API 密钥进行身份验证。

如果您不确定哪些计划包括 Codex 访问，请查看 [定价页面](https://developers.openai.com/codex/pricing)。

3. 3

### 升级

Codex CLI 的新版本会定期发布。请查看 [更新日志](https://developers.openai.com/codex/changelog) 以获取发布说明。要使用 npm 升级，请运行：

npm upgrade 命令

npm i -g @openai/codex@latest复制

Codex CLI 可在 macOS、Windows 和 Linux 上使用。在 Windows 上，可以在 PowerShell 中原生运行 Codex，使用 Windows 沙盒，或在需要 Linux 原生环境时使用 WSL2。有关设置细节，请查看 [Windows 设置指南](https://developers.openai.com/codex/windows)。

如果您是 Codex 新手，请阅读 [最佳实践指南](https://developers.openai.com/codex/learn/best-practices)。

## 使用 Codex CLI

[**交互式运行 Codex** \\
\\
运行 `codex` 以启动交互式终端 UI (TUI) 会话。](https://developers.openai.com/codex/cli/features#running-in-interactive-mode) [**控制模型和推理** \\
\\
使用 `/model` 在 GPT-5.4、GPT-5.3-Codex 和其他可用模型之间切换，或调整推理级别。](https://developers.openai.com/codex/cli/features#models-reasoning) [**图像输入** \\
\\
附加截图或设计规范，以便 Codex 与您的提示一起读取它们。](https://developers.openai.com/codex/cli/features#image-inputs) [**图像生成** \\
\\
直接在 CLI 中生成或编辑图像，并在希望 Codex 对现有资产进行迭代时附加引用。](https://developers.openai.com/codex/cli/features#image-generation) [**运行本地代码审查** \\
\\
在您提交或推送更改之前，让一个单独的 Codex 代理审查您的代码。](https://developers.openai.com/codex/cli/features#running-local-code-review) [**使用子代理** \\
\\
使用子代理并行处理复杂任务。](https://developers.openai.com/codex/subagents) [**网页搜索** \\
\\
使用 Codex 在网上搜索并获取最新的信息以完成您的任务。](https://developers.openai.com/codex/cli/features#web-search) [**Codex 云任务** \\
\\
启动 Codex 云任务，选择环境，并在不离开终端的情况下应用生成的差异。](https://developers.openai.com/codex/cli/features#working-with-codex-cloud) [**脚本化 Codex** \\
\\
通过使用 `exec` 命令脚本化 Codex，以自动化可重复的工作流程。](https://developers.openai.com/codex/noninteractive) [**模型上下文协议** \\
\\
通过模型上下文协议（MCP）为 Codex 提供额外的第三方工具和上下文。](https://developers.openai.com/codex/mcp) [**审批模式** \\
\\
在 Codex 编辑或执行命令之前，选择符合您舒适度的审批模式。](https://developers.openai.com/codex/cli/features#approval-modes)
