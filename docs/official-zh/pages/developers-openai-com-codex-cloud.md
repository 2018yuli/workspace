Codex 是 OpenAI 的编码代理，可以读取、编辑和运行代码。它可以帮助您更快地构建、修复错误和理解不熟悉的代码。使用 Codex 云，Codex 可以在后台（包括并行）处理任务，利用其自己的云环境。

## Codex Web 设置

访问 [Codex](https://chatgpt.com/codex) 并连接您的 GitHub 帐户。这使 Codex 可以处理您存储库中的代码并从其工作创建拉取请求。

您的 Plus、Pro、Business、Edu 或 Enterprise 计划包括 Codex。了解更多关于 [包含内容](https://developers.openai.com/codex/pricing)。某些企业工作区可能需要 [管理员设置](https://developers.openai.com/codex/enterprise/admin-setup)，才能访问 Codex。
