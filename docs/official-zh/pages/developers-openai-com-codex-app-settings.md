复制页面

使用设置面板调整 Codex 应用的行为，文件打开方式和连接工具的方式。从应用菜单打开 [**设置**](codex://settings) 或按 `Cmd` + `,`。

## 一般

选择文件打开的位置，以及命令输出在线程中出现的数量。你还可以要求在多行提示时需要按 `Cmd` + `Enter`，或在线程运行时防止睡眠。

## 通知

选择完成通知何时出现，以及应用是否应提示通知权限。

## 代理配置

应用中的 Codex 代理继承 IDE 和 CLI 扩展的相同配置。使用应用内控件设置常见选项，或者编辑 `config.toml` 进行高级选项。有关更多详细信息，请参见 [Codex 安全](https://developers.openai.com/codex/agent-approvals-security) 和 [配置基础](https://developers.openai.com/codex/config-basic)。

## 外观

在 **设置** 中，您可以通过选择基本主题、调整强调色、背景和前景颜色，以及更改 UI 和代码字体来改变 Codex 应用的外观。您还可以与朋友分享您的自定义主题。

## Git

使用 Git 设置来标准化分支命名，并选择 Codex 是否使用强制推送。您也可以设置 Codex 用于生成提交信息和拉取请求描述的提示。

## 集成与 MCP

通过 MCP（模型上下文协议）连接外部工具。启用推荐的服务器或添加您自己的。如果服务器需要 OAuth，应用将启动认证流程。这些设置同样适用于 Codex CLI 和 IDE 扩展，因为 MCP 配置存储在 `config.toml` 中。有关详细信息，请参见 [模型上下文协议文档](https://developers.openai.com/codex/mcp)。

## 浏览器使用

使用这些设置安装或启用捆绑的浏览器插件，并管理允许和阻止的网站。Codex 会在使用网站之前询问，除非您已将其列入白名单。从黑名单中删除网站将使 Codex 在浏览器中使用它之前再次询问。

请参见 [应用内浏览器](https://developers.openai.com/codex/app/browser) 了解浏览器预览、评论和浏览器使用工作流程。

## 计算机使用

在 macOS 上，检查您的计算机使用设置以查看桌面应用访问和相关偏好的配置。要撤销系统级访问，请在 macOS 隐私与安全设置中更新屏幕录制或辅助功能权限。此功能在推出时不适用于欧洲经济区、英国或瑞士。

## 个性化

选择 **友好**、**务实** 或 **无** 作为您的默认个性。使用 **无** 以禁用个性指令。您可以随时更新此设置。

您还可以添加自己的自定义指令。编辑自定义指令将更新您的 [个人指令在 `AGENTS.md`](https://developers.openai.com/codex/guides/agents-md)。

## 上下文感知建议

使用上下文感知建议，展示您在启动或返回 Codex 时可能想要继续的后续任务。

## 记忆

启用记忆（如有可能），以使 Codex 将过去线程中的有用上下文携带到未来的工作中。请参见 [记忆](https://developers.openai.com/codex/memories) 以获取设置、存储和每个线程控制的详情。

## 存档线程

**存档线程** 部分列出存档聊天的日期和项目上下文。使用 **取消存档** 来恢复线程。
