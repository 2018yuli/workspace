## 入门

Codex 应用程序适用于 macOS 和 Windows。

大多数 Codex 应用程序功能在两个平台上均可用。平台特定的例外将在相关文档中注明。

1. 下载并安装 Codex 应用程序

下载适用于 macOS 或 Windows 的 Codex 应用程序。如果您使用的是基于 Intel 的 Mac，请选择 Intel 版本。

[适用于 macOS（Apple Silicon）下载](https://persistent.oaistatic.com/codex-app-prod/Codex.dmg) [适用于 macOS（Intel）下载](https://persistent.oaistatic.com/codex-app-prod/Codex-latest-x64.dmg)

需要不同的操作系统？

[适用于 Windows 下载](https://get.microsoft.com/installer/download/9PLM9XGG6VKS?cid=website_cta_psi)

[获取 Linux 通知](https://openai.com/form/codex-app/)

2. 打开 Codex 并登录

下载并安装 Codex 应用程序后，打开它并使用您的 ChatGPT 账户或 OpenAI API 密钥登录。

如果您使用 OpenAI API 密钥登录，则某些功能，如 [云线程](https://developers.openai.com/codex/prompting#threads)，可能不可用。

3. 选择项目

选择一个项目文件夹，供 Codex 在其中工作。

如果您以前使用过 Codex 应用程序、CLI 或 IDE 扩展，您将看到过去工作过的项目。

4. 发送您的第一条消息

选择项目后，确保选择 **本地** 以让 Codex 在您的计算机上工作，并向 Codex 发送您的第一条消息。

您可以就项目或您的计算机一般性地询问 Codex。以下是一些示例：

![](https://developers.openai.com/codex/colorcons/brain.png)告诉我关于这个项目的信息！[](https://developers.openai.com/codex/colorcons/gamepad.png)在这个仓库中构建一个经典的贪吃蛇游戏！[](https://developers.openai.com/codex/colorcons/search.png)查找并修复我的代码库中的漏洞，尽量减少高可信度的更改。

如果您需要更多灵感，请探索 [Codex 用例](https://developers.openai.com/codex/use-cases)。
如果您是 Codex 新手，请阅读 [最佳实践指南](https://developers.openai.com/codex/learn/best-practices)。
