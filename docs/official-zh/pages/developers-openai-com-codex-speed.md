## 快速模式

Codex 提供了提升模型速度以增加信用消耗的能力。

快速模式将支持的模型速度提高 1.5 倍，并以比标准模式更高的速率消耗信用。它当前支持 GPT-5.5 和 GPT-5.4，分别以 GPT-5.5 的 2.5 倍和 GPT-5.4 的 2 倍的标准速率消耗信用。

在 CLI 中使用 `/fast on`、`/fast off` 或 `/fast status` 来更改或检查当前设置。您还可以在 `config.toml` 中使用 `service_tier = "fast"` 加上 `[features].fast_mode = true` 来保存默认设置。快速模式可在 Codex IDE 扩展、Codex CLI 和您使用 ChatGPT 登录时的 Codex 应用中使用。使用 API 密钥时，Codex 则使用标准 API 定价，无法使用快速模式信用。

您的浏览器不支持视频标签。

## Codex-Spark

GPT-5.3-Codex-Spark 是一个单独的快速、能力较弱的 Codex 模型，优化用于几乎即时的实时编码迭代。与快速模式不同，快速模式以更高的信用率提高支持的模型速度，而 Codex-Spark 是它自己的模型选择，并具有自己的使用限制。

在研究预览期间，Codex-Spark 仅对 ChatGPT Pro 订阅用户可用。
