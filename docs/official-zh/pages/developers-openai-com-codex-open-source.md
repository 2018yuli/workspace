OpenAI 在开放环境中开发 Codex 的关键部分。该项目在 GitHub 上进行，您可以跟踪进展、报告问题并贡献改进。

如果您维护一个广泛使用的开源项目，或者想提名管理重要项目的维护者，您也可以[申请 Codex for OSS 计划](https://developers.openai.com/community/codex-for-oss)以获取 API 额度、Codex 的 ChatGPT Pro 以及对 Codex 安全的选择性访问。

## 报告问题和请求功能

使用 Codex GitHub 仓库提交有关 Codex 组件的错误报告和功能请求：

- 错误报告和功能请求：[openai/codex/issues](https://github.com/openai/codex/issues)
- 讨论论坛：[openai/codex/discussions](https://github.com/openai/codex/discussions)

提交问题时，请注明您使用的组件（CLI、SDK、IDE 扩展、Codex 网络），并尽可能提供版本信息。
